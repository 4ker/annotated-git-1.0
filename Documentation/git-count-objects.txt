git-count-objects(1)
====================

NAME
----
git-count-objects - Reports on unpacked objects.

SYNOPSIS
--------
'git-count-objects'

DESCRIPTION
-----------
This counts the number of unpacked object files and disk space consumed by
them, to help you decide when it is a good time to repack.
